int MinimaxPlayer::boardScore ( const Board &  board)
protected

Computes this player's heuristic score for a board.

Resulting score depends on who the winner is, as well as the round in which the game ended. Score for a game this player won is always higher than score for a game this player lost.

Resulting score is altered with respect to round number, in a way which prevents the AI player from prematurely giving up a game he can't win, or doing unnecessary moves in a game he can't lose.

std::pair< int, int > MinimaxPlayer::getNextMove ( GameWidget *  game)
virtual

Computes best move for this player in the current game.

Assuming the opponent is playing optimally, the algorithm aims to minimize the maximum possible loss. If several moves yield the same final score, one of them is chosen randomly with a uniform distribution.
